Introduction
This portfolio website is designed to present my professional work and skills in an interactive and visually appealing way. It includes sections such as:

Home: Introduction and welcome message.
Projects: A list of my projects with a search bar for filtering.
Skills: Information about the technologies and programming languages I am proficient in.
Contact: A contact form for getting in touch, along with social media links.
Features
Dark/Light Mode Toggle: Switch between dark and light themes for a personalized viewing experience.
Project Search: A search box that allows users to filter projects by title or description.
Interactive Timeline: A timeline that presents my education and experience in an animated, scrollable format.
Modal Popup: Clicking on a project thumbnail opens a modal with more detailed information and a larger image.
Contact Form: A form for users to get in touch, with basic validation to ensure all fields are filled out properly.
Responsive Design: The site adapts to different screen sizes for an optimal viewing experience on both desktop and mobile devices.
Technologies Used
HTML: Structure of the website, defining sections and content.
CSS: Styling of the website, including layout, colors, fonts, animations, and responsive design.
JavaScript: Provides interactivity, including the dark/light mode toggle, project search functionality, and form validation.
Responsive Design: The website uses media queries to adapt the layout to different screen sizes (mobile, tablet, desktop).

Usage
Once the site is set up, the following features will be available for use:

1. Dark/Light Mode Toggle
Click the "Toggle Dark/Light Mode" button in the navigation bar to switch between dark and light themes. The background and text colors will adjust accordingly.
2. Project Search
In the Projects section, use the search bar to filter through the project cards by title or description. As you type, the list will automatically update to show only the projects that match the search query.
3. Project Modal
Clicking on any of the project images (thumbnails) will open a modal displaying more information about the project. The modal includes a larger image and a description of the project.
4. Contact Form
In the Contact section, fill out the form to send a message. The form validates the input fields (Name, Email, and Message) and shows error messages if any required field is left empty.
